Projects
Out-of-Order RISC-V CPU SystemVerilog
Tomasulo · ROB · Branch Prediction · CSR / Privilege

A cycle-accurate out-of-order superscalar RISC-V processor. Implements Tomasulo's algorithm with a reorder buffer, reservation stations, a load-store queue, a JALR queue, and a complete fetch pipeline with a global history shift register, branch prediction buffer, and return address stack. Handles the full trap and privilege model including exceptions, ecall, and mret. Verified by diff testing vs. Spike.

Operating System on ARMv8 (i.MX 8X) in progress
Barrelfish Multikernel · NXP i.MX 8X · Real Hardware

A fairly complete operating system for NXP's ARMv8-based i.MX 8X hardware built over the course of a semester. Based on the Barrelfish multikernel originally developed at ETH Zurich in collaboration with Microsoft Research. Covers process management, virtual memory, inter-process communication, capability-based security, and device drivers running on real silicon rather than a simulator.

M:N User-Level Thread Library in progress
C · x86-64 Assembly · Work Stealing · Preemption

A user-level M:N threading library written in C. Implements green threads with work-stealing scheduling across multiple kernel threads using Chase-Lev deques, signal-based preemption via SIGPROF, and synchronization primitives built on futexes. Green thread stacks are allocated via mmap with PROT_NONE guard pages for overflow detection. Context switching is implemented in x86-64 assembly saving and restoring callee-saved registers per the SysV ABI.

Distributed Key-Value Store in progress
Consensus · Fault Tolerance · Replication

A distributed key-value store implementing consensus and fault tolerance across multiple nodes. Covers the core tradeoffs in distributed systems design — consistency, availability, and partition tolerance — and the practical engineering of making replication work correctly under failure.

Java Subset Compiler Compilers
Lexing · Parsing · Semantic Analysis · Code Generation

A compiler for a subset of Java covering the full translation pipeline from source to machine code. Implements a scanner and parser, type-checking and semantic analysis passes, and a code generator targeting a register-based intermediate representation.
